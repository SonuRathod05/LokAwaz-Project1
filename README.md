# 🏆 LokAwaz Sentiment Analysis Engine

## 📌 Project Overview
LokAwaz is a civic-tech project designed to analyze Indian political and news headlines. By using Natural Language Processing (NLP), this tool helps citizens understand the emotional tone behind governance updates and policy announcements.

## 🚀 Features
* **Automated Sentiment Scoring:** Uses TextBlob to calculate polarity scores.
* **Sector-wise Analysis:** Categorizes sentiment across Infrastructure, Healthcare, and Economy.
* **Engagement Tracking:** Correlates public upvotes with news sentiment.

## 🛠️ Technology Stack
* **Language:** Python
* **Libraries:** Pandas, TextBlob, Matplotlib, Seaborn
* **Environment:** Jupyter Notebook

## 📊 Sample Insights
- **Infrastructure:** Generally shows 60% positive sentiment due to project approvals.
- **Economy:** Shows higher volatility with a mix of growth reports and protest news.
